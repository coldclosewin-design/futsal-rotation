# Changelog

이 문서는 고객에게 공개할 배포 기준 변경사항만 정리합니다.

버전 규칙:
- `MAJOR`: 큰 구조 변경 또는 호환성 이슈가 있는 변경
- `MINOR`: 기능 추가, 사용자 경험 개선
- `PATCH`: 버그 수정, 문구 수정, 작은 스타일 수정
